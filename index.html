Test The Web

These tests are written to help ensure browsers are doing The Right Thing ®.
It can demonstrate buggy behavior or just prove good support for an existing feature. I try to be as minimal as I can when I write this, so don't expect anything fancy. That just the raw meat.

  • Input events fired by input elements
  • Change events fired by input elements
  • Input events fired by input element when value is set via JS
